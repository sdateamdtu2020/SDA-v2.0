- DATABASE : Postgresql

- dbname = "edsdadb"
- user = "postgres"

### Inside folder database having a file name : perform_edsda.sh

### To perform perform_edsda.sh file : type the follow comment in linux terminal

- 1/ sed -i -e 's/\r$//' perform_edsda.sh
